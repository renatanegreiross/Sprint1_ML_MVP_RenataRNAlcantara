# (Primeiro MVP) Projeto de Predição de Falhas em Máquinas Industriais

Este repositório contém o projeto desenvolvido para a disciplina de Machine Learning (primeira Sprint do curso, então avaliem com moderação por favor).

O objetivo do projeto é construir e avaliar modelos capazes de prever a ocorrência de falhas em máquinas industriais a partir de dados operacionais.

## Arquivos

- **ai4i2020.csv**: base de dados utilizada nas análises e no treinamento dos modelos.
- **MVP_ML_Analytics_RenataR_N_Alcantara_2026_Final.ipynb**: notebook contendo todo o desenvolvimento do projeto, desde a análise exploratória dos dados até a avaliação do modelo final.
